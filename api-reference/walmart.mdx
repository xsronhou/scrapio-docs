title: Walmart
description: Search Walmart and retrieve product details.
---

<Note>
  Amazon and Walmart endpoints require a paid plan (Starter or higher). Without one, these endpoints return `402 plan_upgrade_required`. See [Pricing](https://scrapio.dev/pricing) for details.
</Note>

## Endpoints

- `GET /v1/walmart/search` — Search Walmart and return structured results.
- `GET /v1/walmart/search/crawl` — Queue a Walmart search crawl as an async job.

<Note>
  `GET /v1/walmart/product` is temporarily disabled while we work on
  reliability for individual Walmart product pages. Search and search/crawl
  are unaffected.
</Note>
