title: Amazon
description: Search Amazon, retrieve product details, and fetch offer pricing.
---

<Note>
  Amazon and Walmart endpoints require a paid plan (Starter or higher). Without one, these endpoints return `402 plan_upgrade_required`. See [Pricing](https://scrapio.dev/pricing) for details.
</Note>

## Endpoints

- `GET /v1/amazon/search` — Search Amazon and return structured results.
- `GET /v1/amazon/product` — Retrieve product details by ASIN or URL.
- `GET /v1/amazon/pricing` — Get current offer pricing for a product.
- `GET /v1/amazon/search/crawl` — Queue an Amazon search crawl as an async job.
